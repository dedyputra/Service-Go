This is a special project created collaboratively by developers and contributors to bring an exceptional publishing platform. We are dedicated to making it better every day. --- # First Things First > "Thousands of hours have gone into WordPress, and we’re proud to be a part of it. Thank you for making it part of your world." > — Matt Mullenweg --- # Installation: Famous 5-Minute Install 1. Unzip the package in an empty directory and upload everything. 2. Open `wp-admin/install.php` in your browser: - If it doesn't work, edit `wp-config-sample.php` with your database details and save it as `wp-config.php`. Upload the file and open `wp-admin/install.php` again. 3. Complete the setup and note your credentials (default username: `admin`). 4. Log in via the [login page](wp-login.php). --- ## Updating WordPress ### Using the Automatic Updater 1. Open `wp-admin/update-core.php` in your browser. 2. Follow the instructions provided. ### Updating Manually 1. Back up modified files (e.g., `index.php`). 2. Delete old WordPress files (excluding modified files). 3. Upload the new files. 4. Visit `/wp-admin/upgrade.php` in your browser. --- ## Migrating from Other Systems WordPress can import content from various platforms. For details, visit [WordPress Import Tools](https://wordpress.org/support/article/importing-content/). --- ## System Requirements - **Minimum Requirements**: - PHP version `5.6.20` or greater. - MySQL version `5.0` or greater. - **Recommended**: - PHP version `7.4` or greater. - MySQL version `5.7` or greater, or MariaDB version `10.2` or greater. - Apache `mod_rewrite` module. - HTTPS support. --- ## Online Resources - [The WordPress Codex](https://codex.wordpress.org): Comprehensive documentation for WordPress. - [The WordPress Blog](https://wordpress.org/news/): Latest updates and news. - [WordPress Planet](https://planet.wordpress.org): News aggregator for WordPress-related blogs. - [Support Forums](https://wordpress.org/support/forums/): Community support for questions and troubleshooting. - [WordPress IRC Channel](https://web.libera.chat/#wordpress): Real-time chat with the WordPress community. --- ## Final Notes - Share your ideas, suggestions, or report bugs on the [Support Forums](https://wordpress.org/support/forums/). - Developers can extend functionality using the [Plugin Developer Handbook](https://developer.wordpress.org/plugins/). --- ## License WordPress is free software released under the [GNU General Public License](https://www.gnu.org/licenses/gpl-2.0.html) version 2 or later. --- ## Share the Love ❤️ If you enjoy WordPress, consider: - Sharing it with a friend. - Helping someone set it up. - Supporting by [donating](https://wordpress.org/donate/). --- ### Credits WordPress is the official continuation of [b2/cafelog](http://cafelog.com/) by Michel V.
